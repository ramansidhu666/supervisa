SME Facebook Likebox Sidebar is attractive designed Facebook Likebox Sidebar for Wordpress.

== Description ==

SME Facebook Likebox Sidebar is a facebook sidebar for wordpress website.

Setting up and run this plugin is extremely easy and may be the most easiest Sidebar Plugin for Wordpress available in the directory

Features:

1) Come with three sexy icons to choose from

2) You can customize facebook likebox with many options like border, streams, header and many more.

3) Can define the no. of connections you want to display

4) It use jQuery - will work with all latest jQuery files. By default jQuery not include but you can put it on if need (most case if the site is not running jQuery already)

5) Can add class suffix

and many  more...

== Installation ==

Install the Plugin - top

Login to your wordpress admin - example : http://www.domain.com/wp-admin
go to "Plugins->Installed Plugins" from Menu
Click "Add New" - button at top
Click "Upload" - from the list
Click "Choose File" - button
a windows will open of your computer - browse file - select the package
click "Install Now"

On successful completion of installation you will get a success massage.

Now you have to activate the plugin.

You are all set - Now if you go to your website homepage you probably will see the sidebar up with default options.

Now we have to change options of our preferance.

From Wordpress ADMIN area - go to -> Settings > SME Facebook Likebox Sidebar Configuration

First Option is -

Sidebar Image: Choose the image you like most for your website

Margin Top: Margin from Top Right - Integer value - example: 100

Facebook Page URL: Absolute URL of your facebook like page or business page.
Note: The Facebook Likebox won't work if you put facebook profile URL or facebook group URL instead.

Connections Want to Show: Put the no. of connections you like to display.

Width: Likebox width in px - value must need to be integer - example: 292

Height: Likebox height in px - value must need to be integer - example: 630

Streams: Boolean value - true or false

Color Scheme: You can choose from two available color scheme - light or dark

Show Faces: Boolean value - true or false

Show Header: Boolean value - true or false

Show Border: Boolean value - true or false

Include jQuery: Boolean value - true or false - By default it set to false (You can set it true if your site don't includes jQuery).

Module Class Suffix: It will add a CSS class as you define on facebook likebox
