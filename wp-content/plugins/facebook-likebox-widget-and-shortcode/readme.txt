Facebook Likebox Widget & Shortcode Plugin For WordPress

== Description ==

A versy easy and simple Facebook Likebox Widget & Shortcode Plugin for blog sites.

To activate Facebook likebox widget on you website. Go to widgets area in admin dashboard, find the "Facebook Likebox Widget" named widget and activate in any widget of theme.

== Installation ==
Install Facebook Likebox Widget & Shortcode either via the WordPress.org plugin directory, or by uploading the files to your server.

After activating Install Facebook Likebox Widget & Shortcode, go to widgets menu under appearance menu.

Find the widget named with 'Facebook Likebox Widget'.

Now drag n drop widget in avalable widget areas into your theme.

Finally configure your widget setting and save.

That's it. You're ready to go!
